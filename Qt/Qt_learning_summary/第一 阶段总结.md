### 已掌握的核心能力 (Core Competencies Mastered)

你已经不再是一个初学者。你通过纯代码实践，掌握了构建一个工业级桌面软件骨架所需的几乎全部核心组件。

**1. 👑 Qt 底层对象模型 (The Soul of Qt)**
*   **QObject & 对象树**: 你深刻理解了 Qt 通过“父子关系”自动管理 C++ 内存的“拔萝卜”机制，这是你未来写出无内存泄漏程序的基石。
*   **信号与槽 (Signals & Slots)**: 你掌握了现代 Qt5/6 的 `connect` 语法，理解了其“事件驱动/定闹钟”的异步本质，并解决了 `qOverload` 的重载歧义，这是 Qt 的灵魂。
*   **元对象系统 (Meta-Object System)**: 你通过 `Q_DECLARE_METATYPE` 亲手为自定义类型“办了护照”，使其能在 `QVariant` 中流通，窥见了 `moc` 编译器的冰山一角。

**2. 🧱 核心窗口架构 (The Building Blocks)**
*   **三大窗口基类**: 你清楚 `QWidget` (毛坯房)、`QDialog` (办公室/会议室)、`QMainWindow` (精装房) 的定位和区别。
*   **QMainWindow 布局**: 你掌握了 `QMainWindow` 必须使用 `setCentralWidget` 的核心规则，并用纯代码构建了菜单栏(`QMenuBar`)、工具栏(`QToolBar`)、状态栏(`QStatusBar`)和浮动窗口(`QDockWidget`)。
*   **QDialog 架构**: 你掌握了模态对话框 `exec()` 的阻塞与返回值机制，并能独立封装一个高内聚的自定义对话框（如登录框）。

**3. 🚀 UI 布局与核心工具 (The Craftsmanship)**
*   **布局管理器**: 你已经彻底告别了 `move()` 和 `setGeometry()` 的原始人开发模式，掌握了用 `QHBoxLayout`, `QVBoxLayout`, `QGridLayout` 和“弹簧”(`QSpacerItem`) 来构建自适应、高品质的动态界面。

*   **资源系统 (`.qrc`)**: 你掌握了将图片等资源打包进 `.exe` 的工业级发布方法，解决了软件的移植性问题。
*   **核心数据类型**: 你理解了 `QString` (给人类看), `QByteArray` (给机器看) 的本质区别，并掌握了它们之间以及和 C++ 原生类型 (`int`, `char*`, `std::string`) 之间至关重要的转换方法。

### 技术债务清单 (Technical Debt Backlog)

这是我刻意让你“欠下”的债。它们都属于进阶或高级主题，过早深入会让你陷入泥潭。现在你基础扎实，是时候把它们列入你的“下个版本”开发计划了。

*   **头号技术债：多线程 (`QThread`)**: 我们多次提到“UI线程阻塞陷阱”（如 `QTimer` 或 `QProgressDialog` 的槽函数里不能有耗时操作）。解决这个问题的唯一工业级方案就是多线程。这是你从中级走向高级的**分水岭**。
*   **数据驱动架构：模型/视图 (`Model/View`)**: 我们在讲 `QVariant` 时提到，它是表格 (`QTableView`) 等复杂控件的数据载体。如何让一个有百万行数据的表格不卡顿？答案就是 `Model/View` 架构，这是 Qt 面试的“压箱底”题目。
*   **数据库交互 (`Qt SQL`)**: 现代软件离不开数据库。Qt 提供了强大的 SQL 模块，可以轻松连接 MySQL, SQLite, SQL Server 等。
*   **网络通信 (`QNetworkAccessManager`)**: 如何写一个天气预报软件？如何写一个能从服务器下载文件的程序？这需要网络模块。
*   **高级 UI 美化：样式表 (`QSS`) 与绘图 (`QPainter`)**: 你已经用 `setStyleSheet` 初探了颜色修改。但如何做出像音乐播放器那样酷炫的界面？需要系统学习 QSS (Qt 版的 CSS)。而要画出仪表盘、K线图，则需要动用底层绘图引擎 `QPainter`。
*   **C++ 语法糖：`Lambda` 表达式**: 你已经多次使用 `[=](){}` 作为槽函数。但它内部的捕获机制 (`=`, `&`, `this`) 还有很多门道，值得深究。
