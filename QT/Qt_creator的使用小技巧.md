## 问_1  
那怎么在Qt creator 查项目中所使用到的类？  

## 答_1  
在 Qt Creator 中，你可以通过以下几种方式快速查询和跳转到项目中使用到的类：

### 1. 跳转到定义 (最常用)
*   **操作**：将光标停留在类名（如 `QRegularExpression`）上，按下 **`F2`**。
*   **效果**：Qt Creator 会自动打开该类的头文件。如果是 Qt 官方类，它会带你进入 Qt 库的源码头文件。

### 2. 查找类在何处被使用
*   **操作**：右键点击类名 -> 选择 **"Find Usages"**（查找引用）。
*   **快捷键**：**`Ctrl + Shift + U`**。
*   **效果**：在底部面板会列出整个项目中所有用到该类的地方。

### 3. 使用定位器 (Locator) 全局搜索
*   **操作**：按下 **`Ctrl + K`**。
*   **技巧**：在弹出的输入框中输入 **`c `** (字母 c 后加一个空格)，然后输入类名。
*   **效果**：它会过滤出工作区中所有的 C++ 类。

### 4. 查看类视图 (Class View)
*   **操作**：在左侧边栏的下拉菜单中（默认显示 "Projects"），切换到 **"Class View"**。
*   **效果**：以树状结构列出当前项目中定义的所有类及其成员函数。

### 5. 即时帮助文档
*   **操作**：将光标停留在类名上，按下 **`F1`**。
*   **效果**：会在右侧（或新窗口）直接打开该类的官方帮助文档，查看其所有函数和用法。

### 6. 符号浏览器
*   **操作**：按下 **`Alt + Shift + S`**。
*   **效果**：显示当前文件或项目中的符号列表，方便快速检索。  

## 问_2  
我想使用Qt creator的及时帮助文档，但是我的F1直接和亮度捆绑在一起了，使用不了快捷键  

## 答_2  
针对 F1 键被系统亮度功能占用的问题，你可以尝试以下几种解决方法：

### 1. 使用组合键
在大多数笔记本电脑上，你需要同时按下 **`Fn` + `F1`** 才能触发传统的 F1 功能。

### 2. 在 Qt Creator 中修改快捷键
你可以将帮助文档的快捷键修改为其他不冲突的按键：
1.  打开菜单：**工具 (Tools) -> 选项 (Options)**（在 macOS 上是 Qt Creator -> Preferences）。
2.  选择左侧的 **环境 (Environment)** -> **键盘 (Keyboard)** 选项卡。
3.  在右侧的过滤器框中输入 `ContextHelp`。
4.  在下方找到 `Help` 类别下的 `ContextHelp`。
5.  点击“记录 (Record)”按钮，按下你想要的新快捷键（例如 `Ctrl + H`）。
6.  点击“确定”保存。

### 3. 开启 Fn Lock
检查键盘上是否有 `Esc` 键或专门的 `Fn` 锁定键，按下 **`Fn` + `Esc`** 通常可以切换 F1-F12 的默认行为，使其不再优先触发亮度调节。

### 4. 使用鼠标菜单
如果不想使用快捷键，你可以直接：
*   **右键点击**代码中的类名。
*   选择弹出菜单中的 **Show Help**。

### 5. 使用侧边栏帮助模式
点击 Qt Creator 左侧栏的 **“帮助 (Help)”** 图标，在搜索框中直接输入 `QRegularExpression` 也可以查阅文档。   

## 问_3  
那Qt creator是否允许我直接在帮助mode中搜索对应的帮助文档？  

## 答_3  
是的，**Qt Creator 完全支持在帮助模式（Help Mode）中直接搜索文档**。

你可以通过以下几种方式在帮助界面进行搜索：

### 1. 进入帮助模式
*   点击左侧边栏顶部的**“帮助” (Help)** 图标（通常是一个学士帽图标）。
*   或者按下快捷键 **`Ctrl + 7`** 直接切换到帮助视图。

### 2. 使用“索引” (Index) 搜索 —— 最推荐
在帮助界面的左侧面板中，有一个 **“索引” (Index)** 选项卡：
*   **操作**：在文本框中直接输入类名（例如 `QRegularExpression`）。
*   **特点**：这是最快的方式。当你输入时，列表会自动匹配到对应的类或函数。双击搜索结果即可在右侧查看详细文档。

### 3. 使用“查找” (Search) 搜索
如果你不记得类名，只记得某个关键词：
*   点击左侧面板的 **“查找” (Search)** 选项卡。
*   **操作**：输入你想要查找的关键字（例如 "Regular expression grammar"）。
*   **特点**：它是全文搜索，会搜出所有包含该词的内容，类似于搜索引擎。

### 4. 使用“定位器” (Locator) 直接打开文档 —— 最快途径
如果你甚至不想切换到帮助模式，可以使用顶部的定位器：
*   按下 **`Ctrl + K`**。
*   输入 **`? `** (问号加空格)，然后输入类名。
*   **示例**：`? QRegularExpression`
*   **效果**：它会弹出一个包含匹配文档的列表，回车后会直接打开帮助文档。

### 总结
对于你提到的 `QRegularExpression`：
1. `Ctrl + 7` 进入帮助模式。
2. 在左侧 **Index** 框输入 `QRegularExpression`。
3. 它是非常有用的，因为正则表达式的语法（如 `^`, `$`, `[0-9]`）在文档中都有极详细的说明表格。
